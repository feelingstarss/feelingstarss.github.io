- Lập trình Java, JavaScript
- Lập trình mạng (Network Programming)
- Các công nghệ Web hiện đại

## Kỹ năng

- **Ngôn ngữ:** Java, JavaScript, Python
- **Web:** HTML, CSS, ReactJS, Node.js

- **Tools:** Git, Docker, VS Code
